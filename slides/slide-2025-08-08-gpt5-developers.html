Introducing GPT‑5 for developers
The best model for coding and agentic tasks.

Key Points

  • State-of-the-art coding: 74.9% on SWE‑bench Verified, 88% on Aider polyglot; strong frontend performance.
  • Agentic strength: 96.7% on τ2‑bench (telecom); more reliable tool calling and chaining.
  • New API controls: verbosity (low/medium/high), minimal reasoning_effort, and custom tools with plaintext/CFG.
  • Models and pricing: gpt‑5 / mini / nano; gpt‑5 at $1.25/1M in, $10/1M out (others cheaper).
  • Long context: up to 400k total tokens (272k input, 128k reasoning+output); improved retrieval accuracy.

OpenAI released GPT‑5 on the API platform, delivering SOTA performance for coding and agentic tasks with stronger tool use and long‑context retrieval.
